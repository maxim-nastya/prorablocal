# Бэкенд-сервер "Прораб" (Архив)

**ВНИМАНИЕ: Этот бэкенд-компонент больше не используется.**

Приложение "Прораб" было преобразовано в полностью клиентскую (client-side) версию, которая работает автономно в браузере и хранит все данные в `localStorage`.

Этот код остается в репозитории в качестве архива и примера возможной серверной реализации. Он не является необходимым для работы текущей версии приложения.
